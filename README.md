## How to install

1. Clone this repositoy:

```bash
git clone https://github.com/VKrishan04/Ziolabs.git
```

2. Go into project

```bash
cd shadcn-landing-page
```

3. Install dependencies

```bash
npm install
npm audit fix
```

4. Run project

```bash
npm run dev
```
